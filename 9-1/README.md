Практична робота 9.1 – Варіант 1

Опис проєкту
Даний додаток реалізує менеджер користувацьких налаштувань
у Flutter з використанням локального зберігання через
SharedPreferences.
Користувач може змінювати налаштування профілю, вигляду
інтерфейсу, сповіщень та улюблених категорій.
Усі дані автоматично зберігаються та відновлюються
після перезапуску додатку.

Використані технології
Flutter
Dart
SharedPreferences (^2.2.2)

Як запустити
Клонувати репозиторій
git clone <your-repo-link>
Перейти в папку
cd project
Встановити залежності
flutter pub get
Запустити
flutter run
